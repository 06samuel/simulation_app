Dokumentasi Aplikasi

Panduan Penggunaan Dasbor Green Valley

Selamat datang di dokumentasi aplikasi Green Valley. Halaman ini berisi panduan penggunaan fitur-fitur utama aplikasi yang dibangun menggunakan HTML, CSS, dan JavaScript native (vanilla) sebagai Single Page Application (SPA).

Fitur Utama

  • Navigasi SPA: Berpindah antar halaman (Dasbor, Galeri, Manajemen, Dokumentasi) tanpa reload browser penuh, menggunakan routing berbasis hash URL.
  • Manajemen State: State aplikasi (tema, data booking, statistik) dikelola secara terpusat menggunakan objek JavaScript dan disimpan di Local Storage untuk persistensi.
  • Dasbor Interaktif: Menampilkan statistik kunci, grafik tingkat hunian (bar chart interaktif), daftar pemberitahuan, dan tabel booking terbaru dengan fungsi Tambah, Edit, dan Hapus (CRUD).
  • Tema Aplikasi: Pengguna dapat mengganti tema antara mode Terang (Light) dan Gelap (Dark). Pilihan tema disimpan di Local Storage.
  • Sidebar Responsif: Sidebar navigasi dapat disembunyikan di layar kecil dan ditampilkan kembali dengan tombol toggle, lengkap dengan overlay.
  • Notifikasi Toast: Sistem notifikasi non-blocking untuk memberikan feedback kepada pengguna atas aksi yang dilakukan.
  • Desain Responsif: Tampilan aplikasi menyesuaikan dengan berbagai ukuran layar perangkat (desktop, tablet, mobile) menggunakan Flexbox dan Grid System CSS.
  • Komponen Modular: Styling CSS dan logika JavaScript diorganisir secara modular untuk kemudahan maintenance dan reusability.
  • Aksesibilitas (Dasar): Penggunaan atribut ARIA untuk meningkatkan pengalaman pengguna dengan teknologi asistif.

Struktur Kode

Aplikasi ini dibangun sebagai monolith native tanpa framework/library eksternal (kecuali Font Awesome untuk ikon) untuk memenuhi kebutuhan spesifik akan kode vanilla.

  • HTML: Menggunakan struktur semantik HTML5 dengan penekanan pada aksesibilitas (atribut ARIA).
  • CSS: Menggunakan variabel CSS untuk tema dan konsistensi, Flexbox dan Grid untuk layout, serta styling komponen kustom yang responsif dan modern.
  • JavaScript:
    • Utils: Modul untuk fungsi-fungsi utilitas umum (sanitasi input, format data, pembuatan ID unik).
    • AppState: Objek untuk manajemen state aplikasi secara global, termasuk penyimpanan dan pemuatan dari Local Storage.
    • DOMManager: Objek yang bertanggung jawab untuk semua manipulasi DOM dan rendering elemen UI berdasarkan state aplikasi.
    • App: Modul utama yang menginisialisasi aplikasi, menangani event listener, logika bisnis, dan mengkoordinasikan interaksi antar modul.

Penggunaan Umum

  • Navigasi Antar Halaman: Gunakan menu di sidebar kiri. URL akan berubah (menggunakan hash) dan konten halaman akan diperbarui tanpa reload.
  • Penggantian Tema: Klik ikon bulan (untuk mode Gelap) atau matahari (untuk mode Terang) di header kanan atas.
  • Manajemen Booking: Di halaman Dasbor, tabel "Booking Terbaru" memungkinkan Anda menambah, mengedit, atau menghapus data booking melalui tombol-tombol aksi yang tersedia.
  • Interaksi Grafik: Arahkan kursor ke bar pada grafik "Tingkat Hunian" untuk melihat detail nilai.
